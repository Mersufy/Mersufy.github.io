MersufBT Commands

Join and Part:

Command Name Aliases Cooldown Description
?Join 3 Seconds Join the user's channel.
?Part 3 Seconds Leave the user's channel.
?Ping 3 Seconds Pong.

Game:

Command Name Aliases Cooldown Description
?Fruit 1 Seconds A game where you randomly collect fruits.
?Sell [amount] 1 Seconds Sell a specified number of each fruit and earn coins.
?Gamba [amount] 1 Seconds Gamble with your fruits. Chance to gain or lose fruits.
?Stats 1 Seconds Show your current fruits, coins, and ranking.
?Leaderboard 1 Seconds Show the top 10 users by coins.
?Rankup 1 Seconds Upgrade your rank by spending coins.

7tv:

Command Name Aliases Cooldown Description
?7tvu [username] 3 Seconds Displays information about a 7TV user.
?7tsa [username] 3 Seconds Get the 7TV subscription age of a user.
?Set [username] 7tvset 3 Seconds Shows emote set info
